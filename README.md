## Features
 * 简单的去广告功能
 * 屏蔽网站上的推荐/热搜/热榜等内容
 * `workTimeGuard` 针对指定网站，工作时间遮罩提示不要打开，有允许访问按钮 √
 * `pageTimeTracker` -  统计浏览时长和次数，并可以查看 - TODO 不要用localStorage存储数据
 * 自由编辑页面 - 加开关 `contenteditable="true"`
